# Faculty Editor DevTools 3 - Lingua Italiano IT

## Cos'è Faculty Editor DevTools?

Faculty Editor DevTools è uno strumento utile per:

- Debugging
- Tracciamento delle versioni _con Git_
- Esecuzione di comandi nel terminale

Ed è per questo che gli sviluppatori lo usano.

## Novità

- **Disponibile in altre lingue** ora grazie a DevTools 3, Devtools è Disponibile in Inglese, Italiano, Spagnolo e Tedesco - _Nessun'altra novità_

## Guida rapida

### 1 - Base: Configurare il repository Git

Apri **DevTools** e fai clic sul pulsante "Configurare".

- Configura il nome
- Configura l'email
- Configura il repository Git

>Usa i repository Git locali, altrimenti verrà visualizzato un errore 404 "Bad Request".

### 2 - Base: Commit del progetto

Supponendo che il file si chiami `hello-world.js`, procediamo con il commit.

- Clicca su "Commit"
- Inserisci il messaggio di commit (in questo esempio, "Commit example for hello-world.js")
- Commit completato!

### 3 - Terminale: Comandi base

Digita nel terminale:

```Bash
pwd
```

E clicca su "Eseguire".

L'output sarà:

```ShellOutput
FacultyEditorExecutingPath
```
