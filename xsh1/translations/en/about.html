📜 About Faculty Editor

Installed version: Version 1.0.4 – May 11, 2026

Faculty Editor is a simple, minimalistic and intuitive text editor. It is designed to be easy to use and provide a smooth editing experience.

Faculty Editor, despite being quite advanced, as you can see, is built with HTML, CSS and JavaScript. We use these technologies to ensure simplicity and cross-platform compatibility.

I, the author, although I use VS Code for development, am very proud of how Faculty Editor has turned out and hope you will appreciate it.
