# Faculty Editor DevTools 3 - Idioma Español ES

## ¿Por qué las herramientas para Faculty Editor DevTools?

Las herramientas para desarrolladores del editor de profesores son útiles para:

- Depuración
- Seguimiento de versiones _con Git_
- Ejecución de comandos en la terminal

Por eso las usan los desarrolladores.

## ¿Qué hay de nuevo?

- **Disponible en otros idiomas**: ahora, gracias a DevTools 3, DevTools está disponible en inglés, italiano, español y alemán  
- _Ninguna otra novedad_

## Guía rápida

### 1 - Base: Configurar el repositorio Git

Abre las **Herramientas para desarrolladores** y haz clic en el botón "Configurar".

- Configurar el nombre
- Configurar el correo electrónico
- Configurar el repositorio Git

>Usar repositorios Git locales; de lo contrario, se producirá un error 404 "Solicitud incorrecta".

### 2 - Base: Confirmación del proyecto

Suponiendo que el archivo se llama `hello-world.js`, procedamos a confirmar los cambios.

- Haga clic en "Commit"
- Inserte el mensaje de confirmación (en este ejemplo: "Ejemplo de confirmación para hello-world.js")
- ¡Confirmación realizada!

### 3 - Terminal: Comandos básicos

Escriba en la terminal:

```Bash
pwd
```

Y haga clic en "Ejecutar".

La salida es:

```ShellOuttut
FacultyEditorExecutingPath
```
