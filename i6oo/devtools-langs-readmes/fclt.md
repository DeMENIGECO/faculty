# Faculty Editor DevTools 2 - Language English EN-FCLT

## Why is Faculty Editor DevTools?

The Faculty Editor DevTools are useful tools for:

- Debugging
- Tracking versions _with Git_
- Running commands in the terminal

And that's why developers use them.

## Rapid Guide

### 1 - Base: Configurate Git Repository

Open **Devtools** and click "Configure" button.

- Configure name
- Configure email
- Configure Git repo

>Use locals Git Repositoryes, otherwise it will give a 404 "Bad request" error.

### 2 - Base: Commit of project

Assuming the file is called `hello-world.js`, let's commit.

- Click "Commit"
- Insert commit message (for this example, "Commit example for hello-world.js".)
- Commit done!

### 3 - Terminal: Base commands

Digit onto terminal:

```Bash
pwd
```

And click "Execute."

The output is:

```ShellOuptut
FacultyEditorExecutingPath
```
