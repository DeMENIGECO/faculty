# Faculty Editor DevTools 3 - Deutsche Sprache GE

## Was ist Faculty Editor DevTools?

Faculty Editor DevTools ist ein nützliches Tool für:

- Debugging
- Versionsverfolgung _mit Git_
- Ausführung von Befehlen im Terminal

Deshalb verwenden Entwickler es.

## Neuigkeiten

- **In anderen Sprachen verfügbar**: Dank DevTools 3 ist DevTools jetzt auf Englisch, Italienisch, Spanisch und Deutsch verfügbar  
- _Keine weiteren Neuigkeiten_


## Schnellstart-Anleitung

### 1 - Basis: Git-Repository konfigurieren

Öffne **DevTools** und klicke auf die Schaltfläche „Konfigurieren“.

- Namen konfigurieren
- E-Mail konfigurieren
- Git-Repository konfigurieren

> Verwende lokale Git-Repositories, sonst wird ein Fehler 404 „Bad Request“ angezeigt.

### 2 - Basis: Projekt-Commit

Angenommen, die Datei heißt `hello-world.js`, dann führen wir den Commit durch.

- Klicke auf „Commit“
- Gib die Commit-Nachricht ein (in diesem Beispiel: „Commit-Beispiel für hello-world.js“)
- Commit abgeschlossen!

### 3 - Terminal: Grundbefehle

Gib im Terminal ein:

```Bash
pwd
```

Und klicke auf "Ausführen".

Die Ausgabe lautet:

```ShellOutput
FacultyEditorExecutingPath
```
